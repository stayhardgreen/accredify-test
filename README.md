## System requirement
- PHP 8.1.0 or higher;
- Mysql 8.1

## Usage
- Clone the repo
- ``composer install``
- ``cd {directory_name}``
- ``cp .env.test .env``
- set up as Database setting into env
- ``php artisan migrate``
- ``php artisan test``
